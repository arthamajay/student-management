# College Event Management System (FrontendTasks/Task-3)

Responsive college event website built with **Bootstrap**.

## Pages
- `home.html` — Home, hero banner, carousel, highlights, Register CTA
- `events.html` — 6+ event cards with images, name, date, venue, Register button
- `register.html` — Registration form (Student Name, Roll Number, Branch, Email, Mobile Number, Event Selection) + success alert
- `schedule.html` — Responsive schedule table (Time • Event • Venue)
- `gallery.html` — Photo grid + Bootstrap modal
- `faq.html` — FAQ accordion
- `contact.html` — Contact form + coordinator details + social links

## Technologies
- HTML5
- CSS (custom in `assets/css/styles.css`)
- Bootstrap 5 (CDN)


## How to run
Open `home.html` in a browser.

Note: Pages are static (no custom JavaScript; forms/buttons do not submit anywhere).
